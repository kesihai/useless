#### 维权内容
1. 门楼要做，旁边要有放快递的地方和保安室.
2. 门窗更换，合同写的是铝合金窗户，实物是半铝合金和朔钢材料.
3. 房屋公摊面积占的百分比.
4. 绿化面积占百分之.
5. 停车场是否符合小区设计要求.
6. 园建休闲能否满足小区人员活动空间.
7. 门禁.
8. 必须人车分流.
9. 小区地面不能停车子，小孩不安全.
